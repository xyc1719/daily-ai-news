## 1. ZR-0：用密集"具身思维链"监督训练 VLA，首次实现跨 embodiment 推理复用

- **来源**: 人大/RUC-KBReasoning 团队
- **论文**: *Training Vision-Language-Action Models with Dense Embodied Chain-of-Thought Supervision*  
  arXiv: [2606.30552](https://arxiv.org/abs/2606.30552) | GitHub: [RUCKBReasoning/ZR-0](https://github.com/RUCKBReasoning/ZR-0)
- **简述**: 提出 **ZR-0**（2.6B 参数），核心创新是用**密集具身思维链（ECoT）**标注取代传统的行为克隆监督。架构采用 System 1/2 双流设计：预训练 VLM（System 2）在训练时生成结构化 ECoT 推理，Diffusion Transformer 动作专家（System 1）通过 flow matching 生成连续动作 chunk；关键设计——二者通过 **masked cross-attention** 耦合，推理时 ECoT 完全跳过，零性能损失。预训练数据集 **ProcCorpus-60M** 含 40 万条轨迹、~1000 小时视频，96.8% 帧有 ECoT 标注。在 LIBERO（单臂）、RoboTwin 2.0（双臂）、RoboCasa GR-1（人形）三个仿真平台 + xArm 真机验证，均表现强劲。
- **为什么值得关注**:
  - **VLA 训练方法论的范式级贡献**：首次证明可以将 VLM 的 CoT 能力系统性地迁移到机器人动作生成中，且推理时零额外开销——这对实际部署至关重要。
  - **跨 embodiment 泛化** 是该领域的核心难题，ZR-0 的方案（在 VLM 隐空间对齐而非动作空间对齐）提供了一个干净可扩展的思路。
  - **完全开源**（模型权重 + 代码 + 60M 帧数据集），可复现性极强，适合作为 VLA 研究的 baseline。
- **技术启发**: ECoT 的"推理时跳过"设计思路或可借鉴到视频理解中——用 System 2 的视频推理能力在训练时蒸馏 slow features，推理时仅用 fast stream，兼顾精度与速度。

## 2. 美团 LongCat 三连发：原生多模态 + 具身动作基准 + 视频世界模型评估

- **来源**: 美团 LongCat 团队
- **发布内容**:
  - **LongCat-Next**: 原生视觉+语音多模态模型，将视觉和语音作为原生组件（非外设输入），面向物理世界 AI 场景。开源包含核心模型 + 专用离散分词器。
  - **LARYBench**: 具身动作表征的系统化评估框架，被比作"具身动作的 ImageNet"。关键发现：**通用视觉模型在动作泛化和控制精度上优于专门的具身 AI 动作专家模型**——复杂的具身动作表征可以从大规模人类视频数据训练中自然涌现。
  - **WBench**: 首个系统化的**多轮交互式视频世界模型基准**，涵盖从月球环境到未来城市等多种场景，被比作"世界模型的 CT 扫描仪"，诊断从被动观察到主动多阶段交互转换时的技术瓶颈。
- **为什么值得关注**:
  - 这是国内大厂在**视频 + 具身 + 世界模型**交叉领域最系统化的一次开源发布，覆盖模型、数据和评估三个层次。
  - LARYBench 的核心发现——"通用视觉 > 专用具身"——对研究生而言是一个**重要的研究信号**：与其从头训具身专用模型，不如思考如何更好地利用视频预训练。
  - WBench 填补了交互式视频世界模型评估的系统化空白，是发论文可直接使用的 benchmark。
- **技术启发**: LARYBench 的结论暗示**大规模视频预训练可能是具身智能的 silver bullet**——如果你的研究方向是视频+具身交叉，可以重点探索"如何让视频预训练更好地下游服务于具身任务"。

## 3. T²VLA：用置信度驱动 Test-Time RL，VLA 无需外部奖励即可自我提升

- **来源**: Siyao Chen 等
- **论文**: *Trust Your Instincts: Confidence-Driven Test-Time RL for Vision-Language-Action Models*  
  arXiv: [2606.29892](https://arxiv.org/abs/2606.29892)
- **简述**: 提出 **T²VLA**，一个架构无关的 test-time RL 框架，不需要外部环境奖励。核心发现：在离散动作 VLA 中，生成置信度更高的轨迹成功率显著更高。基于此，T²VLA 使用**轨迹与高置信度专家演示的相似度作为内在奖励信号**，并通过 Confidence-Driven Dual Expert Bootstrapping（Local Pseudo-Expert 探索 + Global Expert Pool 稳定）实现自我提升。在 LIBERO 和 RoboTwin 上超越监督 baseline，逼近 oracle RL 性能，兼容 OpenVLA-OFT 和 π 系列两种 VLA 范式。
- **为什么值得关注**:
  - 延续了上周 **E-TTS**（具身 Test-Time Scaling）的趋势线——说明"推理时算力提升具身模型"不是孤立现象，而是一个正在形成的子方向。
  - "无需外部奖励"的设计对现实场景极具价值：真实机器人部署时往往无法获得密集奖励信号，置信度作为内在信号提供了一条实用路径。
  - 架构无关性意味着可以即插即用到其他 VLA 模型上，适合做 follow-up 实验。
- **技术启发**: 能否将 T²VLA 的"置信度→内在奖励"思路扩展到**视频理解 Agent**上？例如，让视频分析 Agent 根据自身视觉理解的置信度决定是否需要"再看一遍"或请求人类反馈。

## 4. Scaling the Horizon, Not the Parameters：35B Agent 达到万亿参数模型性能

- **来源**: Intern Science（上海人工智能实验室关联）
- **论文**: *Scaling the Horizon, Not the Parameters: Reaching Trillion-Parameter Performance with a 35B Agent*  
  HuggingFace Daily Papers 热榜，06-30 精选
- **简述**: 提出一种 Agent 缩放的新思路：**不增加模型参数，而是扩展 Agent 的"视野"（horizon）**——即通过更长的上下文窗口和更好的信息检索机制，让 35B 参数的 Agent 在复杂多步任务中达到万亿参数模型的性能水平。核心是利用高效的记忆管理和上下文压缩技术，使小模型能够在长周期任务中保持连贯推理。
- **为什么值得关注**:
  - 这是对"scaling law = 更大模型"的直接挑战，对资源有限的研究生群体意义重大：**你不需要 A100 集群也能做 SOTA Agent 研究**。
  - Efficient context management 是 Agent 领域的关键瓶颈，该工作的"扩展视野"思路为 Agent 记忆机制研究提供了新方向。
  - 与视频 Agent 天然相关：长视频理解本质上就是"扩展时间视野"——能否将该思路迁移到长视频 Agent 中？
- **技术启发**: "扩展视野而非参数"的思路可以直接迁移到**长视频理解 Agent**设计中——不训更大的视频模型，而是优化 Agent 对长视频的记忆管理和信息检索策略。

## 5. 桥接视频问答与视频引导 Agent 任务：广义关键帧提取

- **来源**: 5 位作者（HuggingFace Daily Papers 06-30）
- **简述**: 提出通过**广义关键帧提取**来连接视频问答（VideoQA）和视频引导的 Agent 任务。传统上 VideoQA 和 Agent 任务是分离的，该工作发现一个统一的 keyframe extraction 模块可以同时服务于两种下游任务，在 VideoQA 准确率和 Agent 任务完成率上均有提升。
- **为什么值得关注**:
  - **直接落在"视频 + Agent"交叉点上**，正是你关注的领域。该工作表明视频理解和 Agent 行为之间可以通过信息压缩（关键帧）形成统一接口。
  - 这是一个**可快速跟进的方向**：关键帧提取是一个经典问题，但与 Agent 的结合还远未被充分探索——可以做的工作包括：动态关键帧选择策略、Agent 反馈驱动的自适应关键帧、多粒度时间表示等。
  - HuggingFace 当日精选说明社区对该交叉方向的关注度在上升。
- **技术启发**: 一个直接可做的方向——设计一个**Agent 反馈驱动的自适应关键帧选择器**，让 Agent 在任务执行过程中主动决定"我需要看哪些帧"，而不是被动接收固定的关键帧序列。这在具身场景（机器人需要根据任务动态调整视觉注意力）尤其有意义。
