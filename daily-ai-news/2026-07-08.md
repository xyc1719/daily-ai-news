## 1. InternVLA-A1.5：理解、潜在预见与动作的统一 VLA 模型

**机构/作者**：Shanghai AI Lab / OpenGVLab（29位作者，Haoxiang Ma, Tai Wang 等）  
**论文**：[arXiv:2607.04988](https://arxiv.org/abs/2607.04988) | [HuggingFace](https://huggingface.co/papers/2607.04988)

**做了什么**：InternVLA-A1.5 在 VLM backbone 上构建机器人操控策略，核心创新是将"未来预测"重新定义为**潜在空间查询问题**——用少量可学习的 foresight tokens 在冻结的预训练视频生成模型监督下，将任务相关的未来压缩为紧凑潜在编码。inference 时丢弃视频分支，保持实时控制。在 120 万机器人轨迹 + 300 万多模态样本上预训练，在全部 6 个仿真基准上取得最佳综合结果，并在真实世界展现出最强的组合泛化能力（held-out instruction bindings）。

**为什么重要**：
- **三方向直接交叉**：这篇论文天然站在视频分析（预训练视频生成模型作为动力学先验）× 具身智能（机器人操控策略）× Agent（任务规划与长程执行）的交汇点上
- **方法论创新**：不学习像素级视频生成，而是利用冻结视频生成器的世界模型先验监督潜在空间中的"预见"能力，这是**视频模型能力向具身智能迁移**的优雅方案
- **可复现性强**：开源框架 InternVLA 系列有完整代码和模型权重，适合作为研究的 baseline

**技术启发**：能否将视频生成模型的"预测下一帧"能力泛化为 Agent 的"预测下一步状态"——不仅是视觉状态，还包括任务进度、物体关系等语义层面的状态预测？这对 Agent 的长程规划和自省（self-reflection）有重要意义。

## 2. iFLYTEK-Embodied-Omni：大脑-小脑协作的统一具身基础模型

**机构/作者**：科大讯飞（Yuan Zhang, Jia Pan 等 15 位作者）  
**论文**：[arXiv:2607.02542](https://arxiv.org/abs/2607.02542)

**做了什么**：提出统一的 Omni 框架，在同一模型中联合建模视觉（视频+图像）、语言和动作。核心是"大脑-小脑协作"架构：VLM + 视频生成模型组成高层**大脑**（指令理解、任务规划、进度追踪、未来视觉状态预测），动作生成模型作为低层**小脑**（将规划的子目标直接转化为可执行动作序列）。四个阶段渐进训练：先分别训练 VLM、VGM、AGM，再联合微调。

**为什么重要**：
- **国产团队**的系统性工作，表明国内在具身基础模型方向有完整的自主研发能力
- **架构设计哲学**值得关注：将"思考"与"执行"解耦，类似 System 1/System 2 的分层思想，是当前具身智能的主流趋势
- **视频理解+视频生成+动作生成**的统一建模，对于研究"感知-预测-规划-行动"闭环非常有参考价值

**技术启发**：大脑-小脑架构可以推广到更泛化的 Agent 场景——不是所有 Agent 都需要端到端地从感知到动作，高层语义规划与低层执行分离的设计可能更适合长程、开放式任务。

## 3. DELTAVID：用"大家来找茬"训练视频 MLLM 的细粒度时空感知

**机构/作者**：Yankai Yang, Shuo Yang 等  
**论文**：[arXiv:2607.02551](https://arxiv.org/abs/2607.02551)

**做了什么**：提出 DELTAVID 框架，将跨视频"找不同"转化为可训练的感知信号。核心思路：将局部变化识别、时间边界判断、空间证据组织打包为一个 proxy task，训练视频 MLLM 从粗粒度语义理解向细粒度时空证据推理演进。构建 DELTAVID-10K 数据集和 DELTAVID-Bench 基准，在 MMVU、MLVU、Video-MME、VideoHolmes、LVBench、TempCompass 等 8 个主流视频理解基准上取得显著提升。

**为什么重要**：
- **视频理解的"精度瓶颈"**：当前视频 MLLM 已经能做不错的全局语义理解，但在局部时空细节上普遍薄弱。DELTAVID 提供了一种可扩展的监督信号来弥补这个 gap
- **方法论可迁移**：cross-video difference 作为 proxy task 的设计思路，可以迁移到具身智能中的动作差异识别（如不同操作策略的细微差别）、Agent 执行轨迹的异常检测等场景
- **从"看懂"到"看仔细"**：对视频分析方向的研究生来说，这是一个从 benchmark 驱动到能力驱动的思路转变

**技术启发**：在 Agent 场景中，能否用类似的"轨迹对比"（对比成功 vs 失败轨迹的差异）来训练 Agent 的自我纠错能力？

## 4. SVA：冻结 VLA 模型的测试时自适应——树搜索蒸馏为动作评估

**机构/作者**：南京大学（Xinyi Xie, Zican Hu, Chunlin Chen, Zhi Wang, Pichao Wang 等）  
**论文**：[arXiv:2607.03751](https://arxiv.org/abs/2607.03751) | [HuggingFace](https://huggingface.co/papers/2607.03751)

**做了什么**：针对 VLA 模型泛化脆弱的瓶颈，提出 SVA（Search, Value, and Act）框架。关键发现：**诊断性 pass@k 实验表明冻结的 VLA 在输出分布中已经包含高质量行为**（pass@1 成功率 33%，pass@32 达 92%），问题出在动作评估而非动作生成。SVA 先用 MCTS 在仿真中充分探索 VLA 输出分布，将轨迹经验蒸馏为轻量 Q-value 模型；部署时冻结 VLA 生成多个候选动作，evaluator 选择 uncertainty-regularized Q-value 最高的动作，无需仿真器。结果：9B VLA + SVA 超越 27B VLA 7 个点，推理延迟低 27%。

**为什么重要**：
- **测试时扩展（test-time scaling）在 VLA 的首次成功实践**：类比 LLM 的 best-of-N / self-consistency，但针对具身场景的动作空间特性做了专门设计
- **"以小博大"的工程价值**：9B 模型 + 轻量 evaluator 胜过大模型，这对资源受限的具身部署场景非常有意义
- **方法论通用性**：SVA 的解耦思路（action proposal ≠ action evaluation）可以推广到更多具身任务和 Agent 框架

**技术启发**：Agent 的"执行能力"和"判断能力"可能是两个可以独立优化的维度。为 Agent 增加一个轻量的"后果评估器"（consequence evaluator），可能比增大模型规模更有效。

## 5. EdgeBench：真实环境 Agent 学习的 Scaling Law 首次揭示

**机构/作者**：字节跳动 Seed（Deyao Zhu, Xin Zhou, Jingyuan Zhang 等 40+ 位作者）  
**论文**：[arXiv:2607.05155](https://arxiv.org/abs/2607.05155) | [HuggingFace](https://huggingface.co/papers/2607.05155)

**做了什么**：分析约 **38,000 小时 Agent 与真实环境交互数据**，覆盖 134 个超长程真实世界任务（科学发现、软件工程、组合优化、专业知识工作、形式化数学、交互游戏等，每个任务至少持续 12 小时连续 Agent 操作），首次发现 Agent 在环境学习中的整体性能遵循 **log-sigmoid scaling law**（R²=0.998），且 Agent 学习速度大约**每三个月翻倍**。公开了 51 个任务和完整评估框架。

**为什么重要**：
- **从"训练 scaling law"到"部署学习 scaling law"**：之前的 scaling law 研究主要针对预训练阶段，EdgeBench 关注的是部署后在真实环境中的持续学习能力——这对 Agent 的实际应用场景更关键
- **Agent 学习速度的量化**：每三个月翻倍的发现为 Agent 能力的进步提供了一个可度量的时间尺度，这不仅是科学发现，也影响产业规划和投资决策
- **任务设计值得借鉴**：134 个任务的覆盖面和 12 小时持续操作的设定，为 Agent 评估提供了一个新的方法论范本

**技术启发**：环境反馈的"有效信息密度"可能是决定 Agent 学习效率的关键变量。在自身研究中，如何设计高信息密度的交互环境（而不仅仅是更多交互轮次），是一个值得深入的方向。

## 📊 趋势观察

本周（7/7）论文呈现几个明显趋势：

1. **VLA 进入"精调阶段"**：InternVLA-A1.5 和 SVA 分别从架构设计和测试时优化两个角度提升 VLA 性能，领域正从"能不能做"转向"怎么做得更好、更高效"
2. **视频生成模型作为世界模型先验**：InternVLA 和 iFLYTEK-Embodied-Omni 都利用了预训练视频生成模型的动力学先验——视频生成能力正在成为具身智能的基础设施
3. **Agent 评估方法论的深化**：EdgeBench 的 scaling law 分析和之前 BFCL/tool-calling benchmark 的有效性审计，都在推动 Agent 评估从经验走向科学
